# Library Control System

Console application developed in C# using MySQL for database management.

This project simulates a simple library management system, allowing control of books, users and loans through a console interface.

## Features

* Register books
* Register users
* Borrow books
* Return books
* List all books
* Generate system reports

## Technologies

* C#
* .NET
* MySQL
* DotNetEnv

## Project Structure

This project was intentionally developed without using Object-Oriented Programming (OOP).

The main goal of this version was to:

* Practice procedural logic
* Improve control flow
* Work directly with SQL and database operations
* Strengthen understanding of how the system behaves before applying architectural abstractions

A future version of this project will implement the same system using Object-Oriented Programming (OOP) to improve code organization, scalability, and maintainability.
