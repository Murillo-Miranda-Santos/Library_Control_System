# Library Control System

Console application developed in C# using MySQL for database management.

This project simulates a library management system and demonstrates the evolution from a procedural approach to a layered architecture using Object-Oriented Programming (OOP).

## 📦 Project Structure

- **V1** → Initial version focused on procedural logic and direct SQL manipulation  
- **V2** → Improved version with layered architecture (Service, Repository, Database) and OOP principles  

## 🚀 Features

- Book management  
- User management  
- Loan system  
- Return system  
- Report generation  

## 🧱 Technologies

- C#  
- .NET  
- MySQL  
- DotNetEnv  

## 🎯 Purpose

This project was developed to:

- Strengthen programming logic  
- Understand database interaction  
- Apply Object-Oriented Programming in a real scenario  
- Demonstrate code evolution and architectural improvement
